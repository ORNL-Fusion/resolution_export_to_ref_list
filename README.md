# Convert a publication list exported from Resolution into something useful for an FWP or report
Takes the export of publications from resolution and converts it to a text list of publications in a useful format for reports / FWPs

To use, just go to Resolution and click through the following ...

1. Resolution
2. Publications
3. More Dashboards
4. Select your group / org
5. Select Fiscal or Calendar Year
6. Click on "Publications"
7. Check "Journal" under publication type
8. Export to Excel

That should produce a "report.xlsx" file. 

Place that file in the same directory as this repoistory, then in matlab run

```
generate_bib_from_report
```
